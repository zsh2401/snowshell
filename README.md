# Usage
**Requirements**
| Node.js | NPM |
| ---- | ---- |
| >= 10.15.0 | >= 6.4.1 |

### Get template
```sh
git clone https://github.com/zsh2401/sz-rat
mv sz-rat your-app-name
cd your-name-app
```
### Fetch dependencies
```sh
yarn
```
### Run dev-server
```sh
yarn start
```
You could build the application if you need to release current version.
```sh
yarn build
```
Then the application bundles will appears under the `dist` floder.
